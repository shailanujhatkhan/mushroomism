🍄 Mushroomism: Mushroom Classification Using Machine Learning

About the Project

This project uses machine learning to predict whether a mushroom is edible or poisonous. We trained different machine learning models and compared their performance to find which one worked best.

The project was completed using Python in Google Colab.

------------------------------------------------------------------------------------------------------------------------------------------------

Project Goal

The goal of this project was to:

- Predict whether a mushroom is edible or poisonous.
- Compare different machine learning models.
- Measure how well each model performs.

------------------------------------------------------------------------------------------------------------------------------------------------

Dataset

The project uses the Mushroom Dataset, which contains different features of mushrooms. These features are used to predict whether a mushroom is safe to eat or poisonous.

------------------------------------------------------------------------------------------------------------------------------------------------

What We Did

During this project, we:

- Loaded the dataset
- Cleaned and prepared the data
- Converted categorical data into numbers
- Explored the data using charts and graphs
- Split the data into training and testing sets
- Trained different machine learning models
- Compared the results of each model

------------------------------------------------------------------------------------------------------------------------------------------------

Models Used

- Logistic Regression
- Decision Tree Classifier
- Neural Network (MLP)

------------------------------------------------------------------------------------------------------------------------------------------------

Results

Model| Accuracy
Logistic Regression| 54%
Decision Tree| 93%
Neural Network| 99.96%

The Neural Network gave the best results, while the Decision Tree also performed very well.

Evaluation Methods

We evaluated the models using:

- Accuracy

- Confusion Matrix
- ROC Curve
- ROC AUC Score
